Para conectar el agente de IA a Claude, vamos a configuración (*settings*) y a conectores (*connectors*). La URL del servidor mcp es la que se necesita. Podemos usar "añadir conector personalizado" (*add custom connector*). Si se usa la URL completa sin restricciones, aparecerán todos los escenarios disponibles, incluso escenarios de clientes o de otros equipos, lo cual es excesivo.

Hay otra forma: al final de la URL del servidor mcp, se puede añadir `/sse` y luego **números de ID de escenario** (*scenario id numbers*) para limitar lo que aparece para esa conexión. Si se quieren incluir múltiples escenarios, se copia la restricción del ID del escenario y se añade con un *ampersand* (&) y se pega, para incluir el siguiente número de escenario.
